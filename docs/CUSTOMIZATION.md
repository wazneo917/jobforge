# Customization

## Profile
Edit `config/profile.yml` to define your identity, target roles, compensation, and location.

## Sources
Edit `templates/portals.example.yml` to define target companies and search queries.

## Evaluation
Adjust the scoring and archetype logic under `modes/` when the first implementation is added.

## Outputs
Choose whether you want Markdown only, PDF output, or both.

## Automation
You can run JobForge:
- manually
- through cron
- through a batch runner
- through OpenClaw / agent orchestration
